# Legal and no-hack audit

Only pinned public source locations, public direct-hardware observations, and
generated/synthetic project inputs were used. No private PIF, commercial ROM,
proprietary BIOS, title, filename, cartridge ID, or digest was searched, read,
hashed, copied, staged, committed, or packaged.

Product code adds one typed owner and one exact route. It adds no generic bus,
MMIO map, MI bank, RDRAM array, host shortcut, proof-only runtime path, profile
selector, or first-responder implementation.
