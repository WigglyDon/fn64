# Legal and no-hack audit

Only pinned public N64-IPL source/header material, its declared public static
recomp toolchain, repository-generated synthetic bytes, and public Machine APIs
were used. No private PIF, proprietary BIOS, commercial ROM, title, filename,
cartridge ID, or ROM digest was accessed or used to select behavior.

No generic bus/MMIO/map, ranged RDRAM router, module array, calibration engine,
timer, clock, setting, profile, plugin, compatibility rule, or proof-side
execution substitution was introduced.
