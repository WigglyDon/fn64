# Legal and Private-Input Audit

The implementation and proof use repository source, independently generated
synthetic instruction/data bytes, and pinned public source/documentation only.
No private PIF image, proprietary BIOS, private or commercial ROM, cartridge
path, hash, constant, excerpt, or execution was accessed or packaged.

The public N64-IPL source was inspected for relationships and constants; no
external source tree or proprietary firmware bytes are copied into repository
evidence. Synthetic completion does not advance authentic BOOT-2 or establish
game compatibility.
