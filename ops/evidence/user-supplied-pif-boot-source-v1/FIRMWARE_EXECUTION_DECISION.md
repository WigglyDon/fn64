# Firmware execution decision

Minimal IPL1/IPL2 execution was not implemented.

- `LIVE_REPO_FACT` fn64 has one public CPU execution entrance,
  `Machine::step`, and no PIF instruction-fetch owner, PIF device, SI, generic
  bus, or timing system.
- `UNKNOWN` The smallest complete instruction/address/device subset needed to
  execute arbitrary accepted real PIF firmware is not established.
- `INFERENCE` Adding an executor for generated bytes would not validate a real
  firmware path under this packet and would create architecture beyond the
  source-boundary product.
- `WORKER_CLAIM` Execution remains a possible later decision only if
  source-backed materialization cannot preserve authentic causality and a
  bounded source audit proves the required subset.

No second CPU or firmware executor was introduced.
