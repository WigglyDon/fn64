# Legal and no-hack boundary

- Firmware bundled: no
- Firmware downloaded: no
- Firmware searched automatically: no
- Acquisition or dumping instructions: none
- Hidden default path: none
- Private firmware read or hashed: no
- Firmware bytes or words in source/tests/evidence: no
- Firmware-derived table: no
- Copied assembly/disassembly/code: no
- Title, cartridge ID, filename, ROM digest, firmware digest selector: none
- Host/probe-authored SP IMEM: none
- Backing-zero or unknown-byte fallback: none
- Generic bus or generalized memory map: none
- Broad PIF HLE: none
- SDL/window/audio: none
- Compatibility claim: none

`LIVE_REPO_FACT` Test inputs are generated patterns and do not approximate or
reconstruct proprietary firmware.
