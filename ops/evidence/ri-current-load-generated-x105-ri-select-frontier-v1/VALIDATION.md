# Validation

The candidate is validated with exact home-backed Cargo and fixture paths.
Required proof includes format, warnings-denied clippy, nonzero focused tests,
the full Rust forward gate, both existing direct probes, context/fleet/queue
and Markdown-link checks, a fresh clean checkout, and post-integration
canonical repetition.

Exact command logs, tested SHAs, Context-SHAs, storage audit, and generated-path
cleanup are sealed in the Master artifact rather than committed as terminal
wall output.
